Application facilitates a multi-client chatroom 

Steps to run application

1. compile and run the server file
2. copy the port number that the server is running on
3. complie and run the client file with arguments
	
	arguments -> user chat://host:portNumber

	swap out user with a name and portNumber with the port number return by the server file
	swap out host with an address, replace host with localhost to run on your machine
